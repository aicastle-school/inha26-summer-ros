## 1.3. 실습 명령어

### 노드(Node) 관련
```bash
# 실행 중인 모든 노드 확인
ros2 node list

# 특정 노드의 상세 정보 확인
ros2 node info <node_name>
```

### 토픽(Topic) 관련
```bash
# 모든 토픽 확인
ros2 topic list

# 토픽 정보 상세 조회
ros2 topic info <topic_name>

# 토픽에서 메시지 수신 및 확인
ros2 topic echo <topic_name>

# 토픽에 메시지 발행
ros2 topic pub <topic_name> <msg_type> "<msg_content>"

# 토픽 발행 빈도 확인
ros2 topic hz <topic_name>

# 토픽 대역폭 확인
ros2 topic bw <topic_name>
```

### 카메라 제어
```bash
# 카메라 영상 확인
ros2 topic echo /camera/image_raw/compressed

# 팬(Pan) 제어 (-π ~ π)
ros2 topic pub /camera/pan std_msgs/msg/Float64 "{data: 0.5}"

# 틸트(Tilt) 제어 (-π ~ π)
ros2 topic pub /camera/tilt std_msgs/msg/Float64 "{data: 0.3}"
```

### 속도 제어
```bash
# 전진 (linear.x = 속도)
ros2 topic pub /cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0.5, y: 0, z: 0}, angular: {x: 0, y: 0, z: 0}}"

# 회전 (angular.z = 각속도)
ros2 topic pub /cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0, y: 0, z: 0}, angular: {x: 0, y: 0, z: 0.5}}"

# 정지
ros2 topic pub /cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0, y: 0, z: 0}, angular: {x: 0, y: 0, z: 0}}"
```

### 센서 정보 수신
```bash
# IMU 데이터 확인 (50 Hz)
ros2 topic echo /imu

# 오도메트리 정보 확인
ros2 topic echo /odom

# LiDAR 스캔 확인
ros2 topic echo /scan
```

### 서비스(Service) 및 파라미터(Parameter)
```bash
# 사용 가능한 서비스 확인
ros2 service list

# 서비스 타입 확인
ros2 service type <service_name>

# 서비스 호출
ros2 service call <service_name> <service_type> "<request>"

# 파라미터 목록 확인
ros2 param list

# 파라미터 값 확인
ros2 param get <node_name> <param_name>

# 파라미터 설정
ros2 param set <node_name> <param_name> <value>
```

### ROS2 Bag 녹화 및 재생
```bash
# 모든 토픽 녹화
ros2 bag record -a

# 특정 토픽 녹화
ros2 bag record <topic_name1> <topic_name2>

# 녹화된 bag 재생
ros2 bag play <bag_file>

# 녹화 파일 정보 확인
ros2 bag info <bag_file>
```

### 런치(Launch) 실행
```bash
# launch 파일 실행
ros2 launch <package_name> <launch_file>

# 특정 launch 파일의 노드 목록 확인
ros2 launch <package_name> <launch_file> --show-args
```
