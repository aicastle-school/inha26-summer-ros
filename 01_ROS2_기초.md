|------|------|------|------|

| `/camera/pan` | topic | [`Float64`](https://docs.ros2.org/latest/api/std_msgs/msg/Float64.html) | 팬 (rad) |

## 1.3. ROS2 워크스페이스 및 패키지 관리

### 워크스페이스 생성 및 구조

#### 1단계: 워크스페이스 디렉토리 생성

```bash
# 워크스페이스 생성 (관례적으로 ros2_ws 또는 myros2_ws 등)
mkdir -p ~/ros2_ws/src

# 디렉토리 이동
cd ~/ros2_ws
```

#### 2단계: 워크스페이스 구조 확인

```bash
# 전체 구조 확인
tree ~/ros2_ws/

# 기본 구조:
# ros2_ws/
#   ├── src/          # 소스 코드 디렉토리
#   ├── build/        # 빌드 결과물 (빌드 후 생성)
#   ├── install/      # 설치 파일들 (설치 후 생성)
#   └── log/          # 빌드 로그 (빌드 후 생성)
```

### 패키지 생성

#### 1단계: 패키지 생성하기

```bash
# src 디렉토리로 이동
cd ~/ros2_ws/src

# Python 기반 패키지 생성
ros2 pkg create --build-type ament_python my_first_package

# C++ 기반 패키지 생성
ros2 pkg create --build-type ament_cmake my_cpp_package

# 의존성이 있는 패키지 생성
ros2 pkg create --build-type ament_python my_package --dependencies rclpy std_msgs
```

#### 2단계: 생성된 패키지 구조 확인

```bash
# Python 패키지 구조
my_first_package/
  ├── my_first_package/       # 패키지 소스 코드
  │   └── __init__.py
  ├── test/                   # 테스트 코드
  ├── package.xml             # 패키지 메타데이터
  ├── setup.py                # Python 설치 설정
  ├── setup.cfg               # 설정 파일
  └── resource/
```

#### 3단계: 패키지.xml 편집 (필수)

```bash
# package.xml 파일 수정
cd ~/ros2_ws/src/my_first_package
nano package.xml

# 다음 항목 확인 및 수정:
# <name>: 패키지 이름
# <version>: 버전 정보
# <description>: 패키지 설명
# <maintainer>: 관리자 이메일
# <license>: 라이선스 (Apache-2.0, MIT 등)
# <depend> 또는 <build_depend>: 의존성
```

ros2 topic pub /cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0, y: 0, z: 0}, angular: {x: 0, y: 0, z: 0}}"

## 1.5. 실습 예제 코드

### 미니멀 퍼블리셔(Minimal Publisher)

```python
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class MinimalPublisher(Node):
    def __init__(self):
        super().__init__('minimal_publisher')
        self.publisher_ = self.create_publisher(String, 'topic', 10)
        timer_period = 0.5  # seconds
        self.timer = self.create_timer(timer_period, self.timer_callback)
        self.i = 0

    def timer_callback(self):
        msg = String()
        msg.data = 'Hello World: %d' % self.i
        self.publisher_.publish(msg)
        self.get_logger().info('Publishing: "%s"' % msg.data)
        self.i += 1

def main(args=None):
    rclpy.init(args=args)
    minimal_publisher = MinimalPublisher()
    rclpy.spin(minimal_publisher)
    minimal_publisher.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

### PhysiCar 간단한 주행 코드 (전진/후진)

```python
import rclpy
from rclpy.node import Node
from geometry_msgs.msg import Twist
import time

class PhysiCarDriveController(Node):
    def __init__(self):
        super().__init__('drive_controller')
        self.cmd_vel_pub = self.create_publisher(Twist, '/cmd_vel', 10)
        self.get_logger().info('PhysiCar Drive Controller initialized')
        
    def move_forward(self, speed=0.5, duration=3.0):
        """전진"""
        msg = Twist()
        msg.linear.x = float(speed)
        
        start_time = time.time()
        self.get_logger().info(f'Moving forward: speed={speed}')
        while time.time() - start_time < duration:
            self.cmd_vel_pub.publish(msg)
            time.sleep(0.1)
    
    def move_backward(self, speed=0.3, duration=2.0):
        """후진"""
        msg = Twist()
        msg.linear.x = -float(speed)  # 음수 = 후진
        
        start_time = time.time()
        self.get_logger().info(f'Moving backward: speed={speed}')
        while time.time() - start_time < duration:
            self.cmd_vel_pub.publish(msg)
            time.sleep(0.1)
    
    def stop(self):
        """정지"""
        msg = Twist()
        msg.linear.x = 0.0
        msg.angular.z = 0.0
        self.cmd_vel_pub.publish(msg)
        self.get_logger().info('Stopped')
        time.sleep(0.2)

def main(args=None):
    rclpy.init(args=args)
    controller = PhysiCarDriveController()
    
    try:
        # 전진 3초
        controller.move_forward(speed=0.5, duration=3.0)
        time.sleep(1.0)
        
        # 후진 2초
        controller.move_backward(speed=0.3, duration=2.0)
        time.sleep(1.0)
        
        # 정지
        controller.stop()
        
        controller.get_logger().info('Drive sequence completed')
    except KeyboardInterrupt:
        controller.get_logger().info('Interrupted by user')
    finally:
        controller.destroy_node()
        rclpy.shutdown()

if __name__ == '__main__':
    main()
```

### PhysiCar 카메라 데이터 수신

```python
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import CompressedImage
import cv2
from cv_bridge import CvBridge
import numpy as np

class CameraViewer(Node):
    def __init__(self):
        super().__init__('camera_viewer')
        self.subscription = self.create_subscription(
            CompressedImage,
            '/camera/image_raw/compressed',
            self.image_callback,
            10)
        self.bridge = CvBridge()
        self.frame_count = 0
        self.get_logger().info('Camera Viewer Node started. Window will appear shortly.')

    def image_callback(self, msg):
        """카메라 이미지 수신 콜백"""
        self.frame_count += 1
        
        try:
            # CompressedImage를 OpenCV 이미지로 변환
            np_arr = np.frombuffer(msg.data, np.uint8)
            cv_image = cv2.imdecode(np_arr, cv2.IMREAD_COLOR)
            
            # 로그 스팸을 방지하기 위해 100프레임마다 해상도 정보 출력
            if self.frame_count % 100 == 0:
                height, width = cv_image.shape[:2]
                self.get_logger().info(f'Frame {self.frame_count}: {width}x{height}')
            
            # OpenCV를 사용하여 실시간 이미지 출력
            cv2.imshow("PhysiCar Camera View", cv_image)
            cv2.waitKey(1)  # GUI 이벤트 처리 및 화면 업데이트를 위해 필수
            
        except Exception as e:
            self.get_logger().error(f'Error processing image: {e}')

    def destroy_node(self):
        # 노드 종료 시 모든 OpenCV 창 닫기
        cv2.destroyAllWindows()
        super().destroy_node()

def main(args=None):
    rclpy.init(args=args)
    camera_viewer = CameraViewer()
    
    try:
        rclpy.spin(camera_viewer)
    except KeyboardInterrupt:
        camera_viewer.get_logger().info('Node stopped by user')
    finally:
        camera_viewer.destroy_node()
        rclpy.shutdown()

if __name__ == '__main__':
    main()
```

## 1.9. 주요 학습 포인트

### ROS2의 기본 구성요소

1. **노드(Node)**: ROS2의 기본 실행 단위
   - 독립적으로 실행되는 프로세스
   - 다른 노드와 통신 가능

2. **토픽(Topic)**: 노드 간 비동기 통신 채널
   - 퍼블리셔(Publisher): 데이터 발행
   - 서브스크라이버(Subscriber): 데이터 수신
   - 1:N, N:1, N:N 통신 가능

3. **서비스(Service)**: 노드 간 동기 요청-응답 통신
   - 클라이언트: 요청
   - 서버: 응답

4. **파라미터(Parameter)**: 런타임 설정값 관리
   - 동적으로 노드 동작 조정 가능

### PhysiCar 시뮬레이터 활용

- Codespaces 환경에서 브라우저 기반 ROS2 개발
- 카메라, LiDAR, IMU 등 다양한 센서 데이터 수신
- 로봇의 속도 및 조향 제어를 통한 실습

### 실습 추천 흐름

1. **기본 개념 이해**: 노드와 토픽 확인으로 시스템 구조 파악
2. **센서 데이터 수신**: 카메라, LiDAR, IMU를 echo로 확인
3. **로봇 제어**: 속도 명령 발행으로 로봇 제어 실습
4. **능동 센싱**: 카메라 팬/틸트 제어로 방향 제어 경험
5. **데이터 분석**: Bag 파일로 데이터 녹화 및 분석
